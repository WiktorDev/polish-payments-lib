# polish-payments-lib
Polish Payments Providers nodejs library

### Supported payment operators:

* HotPay
* CashBill
* MicroSMS
* DPay
* PayByLink
* Lvlup
* PayNow

### Installation
```bash
npm install polish-payments-lib
```

For more info please visit docs
```
https://docs.ycode.xyz/node-payments
```
